# User Profile Info

get user profile info

**URL** : `/api/profile/`

**Method** : `GET`

**Auth required** : YES

**Permissions required** : NO

## Success Response

**Code** : `200 OK`

**response example:**
```json
{
    "username": "morteza",
    "email": "abc@gmail.com"
}
```

## Error Response

* **Code** : `403 FORBIDDEN`
